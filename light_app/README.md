# light_app
Регулятор освітлення на nRF7002 DK

Проєкт у Visual Studio Code для плати nRF7002 DK.
Регулювання яскравост світлодіода з допомогою потенціометра.
В проєкті ми покажемо як налаштувати ШІМ (PWM) та АЦП.
